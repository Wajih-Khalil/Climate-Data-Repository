# 🌍 Climate Data Repository (CDR)

A full-stack web-based platform designed for environmental analysts and administrators to manage, analyze, and visualize climate data efficiently.

---

## 🚀 Overview

Climate Data Repository (CDR) is a modern data management and analytics system that enables users to:

- Store and manage climate-related datasets
- Perform data analysis and querying
- Visualize environmental trends
- Support decision-making for climate research

This platform is built with a scalable architecture combining a modern frontend, backend services, and database integration.

---

## 🧠 Key Features

### 📊 Data Management
- Upload and manage climate datasets
- Structured storage using relational databases
- Schema-based data organization

### 🔍 Data Analysis
- Query-based data filtering
- Custom SQL-based analytics
- Efficient data retrieval

### 📈 Visualization (Extendable)
- Graph-based insights (future-ready)
- Trend analysis for environmental patterns

### 🔐 Environment Configuration
- Secure environment variable handling using `.env`
- Configurable database connections

---

## 🏗️ Tech Stack

### Frontend
- **Vite** – Fast modern build tool
- **TypeScript** – Type-safe development
- **HTML/CSS/JS** – UI rendering

### Backend
- **Node.js**
- **TypeScript**
- REST API architecture

### Database
- **SQLite** (local development)
- SQL schema support (`oracle_schema.sql` for extensibility)
